## Install

Install dependencies:

```bash
$ yarn
```

or

```bash
$ npm install
```

## Compile

Compile smart contracts and create typechain:

```bash
$ yarn compile
```

or

```bash
$ npm run compile
```

## Test

Test smart contracts against ./tests:

```bash
$ yarn test
```

or

```bash
$ npm run test
```
